Bienvenido al Redimensionador de Imágenes

Cambia de tamaño imágenes JPG, PNG o GIF definiendo nuevas dimensiones de alto o ancho en píxeles. Además, permite añadir etiquetas a las imagenes de forma que, posteriormente, dichas imagenes puedan ser buscadas en la aplicación. En caso de añadir más de una etiqueta a la imagen, esta debe estar separada por ", ".
